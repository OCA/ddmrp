This module allows to restrict a specific route to be used in the
ddmrp stock buffers. This route will be used instead of the default determined by
default.

**Table of contents**

.. contents::
   :local:

Usage
=====

Go to the stock buffer and change the route. Only the routes applicable
for this location or locations above it can be selected.
